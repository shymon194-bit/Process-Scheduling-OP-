# Process Scheduling Calculator

A web-based interactive application built with HTML and JavaScript to calculate and visualize various operating system process scheduling algorithms [137].

## 🚀 Features
* Supports algorithms like: FCFS, SJF, SRTF, Round-Robin, and Priority [137]
* Calculates Turnaround Time (TAT) and Waiting Time (WT)
* Displays an automated Gantt Chart representation

## 🛠️ Built With
* HTML5
* JavaScript (ES6+)
* CSS3
